Solr Autocomplete plugin for jQuery

URL :Mention the url in which apache solr service is running.
ex:-http://localhost:8080/solr/collection/select?q=
remove extra attributes with solr url.		url:http://localhost:8080/solr/collection/select?q=
GET :Mention the get variable , which means from which column you have to serach data from solr collection.
ex1:-http://localhost:8080/solr/collection/select?q=name:search_value
ex2:-http://localhost:8080/solr/collection/select?q=id:search_value		For ex1
get:"name"

For ex2
get:"id"
FIELDS :Mention the field colums which you have to get for each input.If you want to get the ID of autocomplete fields then add "id" to the field. fields:"name,id"
SHOW :SHOW field tells which field should be replace at current input field.
Most prefer name field to replace.		show:"name"
ROWS :Mention the number of rows to receive for each query.
Default row size is 3.		rows:3
THEME :MOST IMPORTANT FEATURE
You can make your autocomplete design fully customizable.Type your html code and design your css .Customize your html design within theme field function.		theme:function(e){
return'
'+e.name+"
"
}
